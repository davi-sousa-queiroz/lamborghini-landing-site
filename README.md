# 🏎️ Lamborghini Website (May 17th - 23rd 2026)

A Lamborghini inspired landing page built with HTML and CSS.

What started as "I'll make a quick car website before bed" somehow turned into a full multi-section project featuring the Huracán, Aventador, custom animations, luxury-inspired design, and enough CSS to make me question my sanity.

Here's the website link: https://davi-sousa-queiroz.github.io/lamborghini-landing-site/

---

## ✨ Features

- Full-screen hero section
- Lamborghini-themed design
- Animated headings
- Hover effects
- Huracán specification showcase
- Aventador spotlight section
- Call-to-action footer
- Built entirely with HTML and CSS

---

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- CSS Animations (Asked ChatGPT for animations because i have not learned them yet)

## 📚 What I Learned

This project helped me improve:

- Flexbox layouts

More importantly, I learned that building projects teaches me way more than endlessly watching tutorials.

---

## 🚧 Challenges

The biggest challenge was making everything look clean while keeping the luxury Lamborghini aesthetic.

I spent way too much time moving things around by a few pixels, changing font sizes, tweaking spacing, and wondering why one element suddenly decided to launch itself across the screen.

Classic CSS.

---

## 🎯 Why I Built This

I'm currently learning frontend development and wanted a project that felt bigger than a simple tutorial exercise.

Cars have always been cool, Lamborghinis are cooler, and building something that actually looks like a real landing page sounded like a fun challenge.
